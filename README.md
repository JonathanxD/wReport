# wReport
A plugin for Sponge to report bad players and start a vote kick (under development).

#### Features

- [x] Report System
- [ ] GUI
- [ ] Vote kick
- [ ] SQL Support

#### Reasons

- [x] Aggression
- [ ] Illegal Program
- [ ] Bug Abuse
- [ ] Flood
- [ ] Spam

#### Action

- [x] Ban
- [ ] Kick
- [ ] Block Chat (all or specific channels if supported)
